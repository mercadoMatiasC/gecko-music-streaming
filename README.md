
# Gecko - Music Streaming Service

Una aplicación de streaming de música, sube tu propia música, crea tus playlists, comparte tus repertorios con el mundo y únete a un canal con todos tus amigos para escuchar una sesión en tiempo real.  

Objetivo: Aprender y demostrar una arquitectura limpia de diseño tanto de estructura de datos como UX/UI.  

Estado: En desarrollo activo.  
Stack principal: Laravel (Rest API) | React.JS (Frontend SPA).  

## Estructura  

/backend   -> Laravel REST API  
/frontend  -> React.JS Application
